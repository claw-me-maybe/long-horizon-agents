Takeaways

1 · Code has runaway uplift

4.2× across 3 rounds. Each iteration surfaces another production concern (tests → auth → state → ops). "Iteration tax" paid in hardening, not rethinking.

2 · Craft letters plateau after round 2

Editor's v1 was partial; v2 finished it with alternatives. v3 ≈ v2. Once the structural argument is complete, further rounds polish prose but don't transform.

3 · "Tight template" ≠ "no uplift"

Discharge summaries grew 2.4× — patient-education has many small literacy levers (pictographs, meal anchors, caregiver roles) that compound. My prior was wrong.

4 · Regulated templates plateau earliest

Chemist = 1.6×. ICH Q2(R2) fixes the outline; iteration refines validation rigor within a fixed skeleton. No new section headings to discover.

5 · Swarms produce portfolios, not consensus

Editor agents picked 3 genuinely different load-bearing diagnoses (architecture / causality / desire). SW dev agents converged (σ ≈ 2%). Useful for comparison; bad for majority-vote.

The uplift curve depends on the artifact's degrees of freedom. Fixed shape → refinement. Open-ended craft → alternatives. Systems code → hardening.

Zooming out · what does this demo actually hint at?

Scale > peak intelligence

Our 2–4× uplift came from running the same model more times — not a smarter model. The value curve runs on agents × rounds × domains, not peak IQ.

Chemist & lawyer plateau fast: once the shape is set, most iteration decisions are local and easy — "did I cover RSD?", "meal anchor or clock time?" A tiny model could make them. But a model has to be in the loop.

Societal-scale transformation looks like trillions of decisions × tiny per-decision intelligence — nanobot swarms, driverless fleets — not a handful of decisions × frontier intelligence.

Stage 1 → Stage 2

Stage 1 · AI immersion. Editor, Lawyer, Chemist, RN all live here — the model writes the whole artifact. This demo is mostly Stage 1.

Stage 2 · AI retreats to cracks. Software Dev v1→v3 is the leading indicator: one dispatch loop became Redis token bucket + HMAC + CircuitBreaker state machine + DLQ + SIGTERM drain — bespoke deterministic code that the model wrote, then stepped out of.

What Stage 2 at scale still needs: AI writing weights the way it writes code. Today: 45 text artifacts on demand. Tomorrow: billions of specialized models on demand — escalated to by bespoke code in the cracks.
