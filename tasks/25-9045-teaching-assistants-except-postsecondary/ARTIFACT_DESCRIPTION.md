# Small-Group Intervention Packet: 4th Grade Fractions (Tier 2) — Task Cards, Scripts, and Data Sheets

## Occupation
- **SOC:** 25-9045
- **Title:** Teaching Assistants, Except Postsecondary

## Scenario
Ms. Alvarez is a paraprofessional/instructional aide at Hillcrest Elementary supporting 4th grade inclusion classrooms. Under the supervision of the general-education teacher and the district's MTSS specialist, she runs a 25-minute daily Tier 2 small-group intervention block for 5 students: 2 with IEPs (SLD in math), 1 with a 504 plan (ADHD), 1 multilingual learner (WIDA 3, Spanish home language), and 1 student flagged by universal screener (STAR Math) with a fractions deficit. The group meets during core math rotations. She needs a 10-day intervention packet she can use independently, consistent with the classroom teacher's Tier 1 fractions unit (equivalent fractions, comparing, and adding with like denominators).

## Inputs
- **Standards (CCSS-Math, Grade 4) — targeted:**
  - 4.NF.A.1 (equivalent fractions using visual models)
  - 4.NF.A.2 (compare fractions with different numerators and denominators)
  - 4.NF.B.3a (add/subtract fractions with like denominators)
- **Tier 1 pacing (from classroom teacher):** Week 1: identifying fractions with area models; Week 2: equivalence; Week 3: comparing; Week 4: adding like denominators.
- **Student profiles (condensed):**
  - Student A (IEP, SLD math): reads at mid-3rd; needs concrete manipulatives, chunked tasks; IEP goal "will identify equivalent fractions using visual models with 80% accuracy"
  - Student B (IEP, SLD math): benefits from verbal rehearsal, struggles with retention across days
  - Student C (504, ADHD): needs movement, visual timer, choice of 2 tasks
  - Student D (MLL, WIDA 3): academic vocabulary is barrier; benefits from sentence frames and cognates (fracción, equivalente)
  - Student E (screener flag): solid number sense, fragmented part-whole understanding
- **Materials available:** Fraction tiles, fraction circles, number-line templates, pattern blocks, dry-erase boards, visual timer, token chart, laminator access.
- **Supervision model:** Classroom teacher provides the Tier 1 lesson; TA delivers packet; both meet Fridays to review data sheets.
- **Compliance note:** TA must NOT create/modify IEP goals or independently diagnose learning needs. Packet must be teacher-reviewed; TA's role is high-fidelity delivery and accurate data collection.

## Artifact specification
An intervention packet designed for **high-fidelity delivery by a paraprofessional**, aligned to the classroom teacher's Tier 1 unit. This is NOT a lesson plan authored by the TA — it is a set of **accommodation/differentiation supports, task cards, scripts, and data sheets** a TA uses under supervision.

Required sections:
1. **Packet overview**: purpose, role boundaries (what TA does / doesn't do), supervising teacher's sign-off box, data-review cadence.
2. **Group roster & goals snapshot** (1 page, no PII in template): student initials, IEP/504/MLL flag, current data point, target skill for the cycle.
3. **10-day routine skeleton** (25 min each): fixed structure so the TA has a predictable script:
   - Min 0-3: warm-up fluency (flashcards or number talk prompt)
   - Min 3-8: "I Do" modeling with manipulative
   - Min 8-18: "We Do" / "You Do" task card practice
   - Min 18-22: independent check-in (1-2 items)
   - Min 22-25: self-rating + exit ticket + cleanup
4. **Task card set (minimum 15 cards)**: differentiated into 3 tiers (concrete, representational, abstract), each with front (task) and back (answer + scripted prompt if stuck). Cards tagged to 4.NF standard.
5. **Verbal scripts** for TA:
   - Modeling script for equivalent fractions with fraction tiles
   - Corrective-feedback script (3 levels: hint -> scaffolded model -> restate and try)
   - Redirection script for off-task behavior (for Student C)
   - MLL language support script (sentence frames, cognate callouts)
6. **Accommodation cards** (one per student, portable):
   - Student A: use fraction tiles always; 3-item problem sets max; visual vocabulary card
   - Student B: end-of-session verbal recap; spaced-retrieval prompts next day
   - Student C: 10-min timer, 2-choice task menu, token for sustained attention
   - Student D: sentence frames ("___ is equivalent to ___ because ___"); Spanish-English cognate list
   - Student E: probing questions that target part-whole misconception
7. **Data collection sheets**:
   - Daily tally sheet: student x skill x % correct
   - Behavior/engagement note column (for 504 student)
   - Friday summary page for supervising teacher review
8. **Decision rules** (pre-approved by supervising teacher): "If Student A scores <60% for 3 consecutive days on 4.NF.A.1, flag for teacher and revert to more concrete task cards." — TA follows, teacher owns decision.
9. **Vocabulary supports**: visual word wall cards (numerator, denominator, equivalent, compare, whole) with icon, definition in student-friendly language, Spanish translation where appropriate.
10. **Family communication insert** (optional, 1-page, shareable by supervising teacher): what we're practicing, one fun at-home activity.

Pedagogical conventions: Concrete-Representational-Abstract (CRA) progression; gradual release; explicit direct instruction; Tier 2 MTSS fidelity; scripted supports for paraprofessional delivery.

## Output format
Single Markdown file, ~1,800-2,600 words. Must include task card templates (labeled Card 1 / Card 2 etc. with front/back content), data-sheet tables, and verbal-script blocks.

## Iteration targets
1. **Role-boundary clarity** — v1 may accidentally have the TA making instructional decisions; re-scope so instructional decisions remain with the teacher.
2. **Script specificity** — corrective feedback needs exact wording, not "provide support".
3. **Student-level differentiation** — each accommodation card must be genuinely different and tied to the student's profile.
4. **Data sheet usability** — if a TA can't complete it in 2 minutes, it's too complex.
5. **CRA progression across 10 days** — task cards should move from concrete toward abstract with clear criteria for advancing.
6. **MLL integration** — vocabulary supports must go beyond translation to include sentence frames and structured talk.

## Success criteria
- Packet is deliverable by a paraprofessional without making instructional decisions.
- Task cards are organized by CRA tier and standard, with scripted hints on the back.
- Each of the 5 student profiles has a distinct, usable accommodation card.
- Data sheets are simple enough to complete during the session and useful for teacher review.
- Decision rules are stated in if-then form with teacher-owner language.
- Vocabulary supports include visuals, student-friendly definitions, and Spanish-English cognate bridges.
- Scripts are written in exact TA-to-student language (not abstract pedagogical prose).

## Scope target
At 15 minutes, v1 should include: packet overview with role boundaries, roster snapshot, 10-day routine skeleton, a starter set of 5-8 task cards across tiers, draft accommodation cards for all 5 students (even if brief), and a data-sheet template. Full 15+ task cards, complete scripts, and family insert are expected to be partial.
