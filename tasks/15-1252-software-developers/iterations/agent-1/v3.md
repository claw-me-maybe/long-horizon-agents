## `README.md`

```markdown
# plaidstone-webhook-dispatcher

Async webhook dispatcher for Plaidstone Financial settlement events (ticket PLD-2147).
Consumes from the Redis stream `settlements:v2` via consumer group
`dispatcher-g1`, signs each payload, and POSTs it to the merchant's registered
URL with per-merchant rate limiting, retries, and circuit breaking.

Built to close the March 3 duplicate-batch incident: at-least-once delivery is
preserved, but duplicates are actively suppressed by idempotency-key dedup in
Redis with a 7-day TTL.

## Guarantees
| Property                   | Mechanism                                               |
| -------------------------- | ------------------------------------------------------- |
| At-least-once delivery     | Redis consumer-group XACK only on 2xx or terminal drop  |
| No duplicate POSTs         | `SET NX EX=604800` on `idempotency_key`                 |
| Per-merchant rate limit    | Redis `INCR` + `EXPIRE` fixed-window token bucket       |
| Bounded retry storms       | Jittered exponential backoff, max 5 attempts            |
| No thundering herd on down merchants | Circuit breaker: 5 consecutive 5xx -> open, 30s half-open |
| Tamper-evident delivery    | HMAC-SHA256, Stripe-style `t=<ts>,v1=<hex>`             |
| Replay protection (verify) | `hmac.compare_digest` + 5-min max-age                   |
| Graceful shutdown          | SIGTERM/SIGINT -> drain in-flight, close pools          |

## Environment variables
| Var            | Default                                    | Notes                               |
| -------------- | ------------------------------------------ | ----------------------------------- |
| `REDIS_URL`    | `redis://redis:6379/0`                     | Stream + dedup + rate-limit backend |
| `DB_URL`       | `postgresql://plaidstone@db/plaidstone`    | `merchant_webhooks` table           |
| `CONCURRENCY`  | `32`                                       | Max in-flight dispatches            |
| `METRICS_PORT` | `9100`                                     | Prometheus `/metrics`               |
| `CONSUMER_ID`  | `worker-1`                                 | Unique per replica                  |
```

```
## Observability
- Structured JSON logs via `structlog` (stdout).
- Metrics on `:9100`:
  - `webhooks_sent_total{merchant_id, outcome}` — outcomes: `ok`, `dedup`,
    `dlq`, `circuit_open`, `inactive`.
  - `webhook_latency_seconds{merchant_id}` — POST round-trip histogram.
  - `webhook_retries_total{merchant_id, outcome}` — outcomes: `5xx`, `4xx`,
    `exception`.

## Operational notes
- The rate limiter is a per-second fixed-window counter. Strict per-second RPS
  is preserved across replicas because the bucket key lives in Redis. Short
  bursts may straddle a second boundary; if you need sliding-window precision,
  swap `TokenBucket.acquire` for a Lua-based leaky bucket — the call sites
  won't change.
- Dead-lettered events land on `settlements:dlq` with `reason` and original
  payload. Replay by `XRANGE settlements:dlq ...` and re-injecting into
  `settlements:v2`.
```

## `dispatcher/signing.py`

```python
"""HMAC-SHA256 signing + timestamp-tolerant verification.

Header format (Stripe-style): `t=<unix_ts>,v1=<hex>`.

The payload that is signed is the literal bytes `f"{ts}.".encode() + body` —
binding the timestamp into the MAC prevents downgrade / timestamp-swap
attacks.
"""
from __future__ import annotations

import hashlib
import hmac
import time
from typing import Final

MAX_AGE_SECONDS: Final[int] = 300  # reject events older than 5 minutes


def _digest(secret: str, ts: int, body: bytes) -> str:
    payload = f"{ts}.".encode() + body
    return hmac.new(secret.encode("utf-8"), payload, hashlib.sha256).hexdigest()


def sign(secret: str, body: bytes, timestamp: int | None = None) -> tuple[str, str]:
    """Return `(ts_str, header_value)` for the given body."""
    ts = timestamp if timestamp is not None else int(time.time())
    return str(ts), f"t={ts},v1={_digest(secret, ts, body)}"


def _parse_header(header: str) -> tuple[int, str]:
    # Tolerant of whitespace, intolerant of malformed tokens.
    parts: dict[str, str] = {}
    for chunk in header.split(","):
        k, _, v = chunk.strip().partition("=")
        if not k or not v:
            raise ValueError("malformed signature header")
        parts[k] = v
    return int(parts["t"]), parts["v1"]


def verify(
    secret: str,
    body: bytes,
    header: str,
    *,
    now: int | None = None,
    max_age: int = MAX_AGE_SECONDS,
) -> bool:
    """Constant-time HMAC verification with a freshness window.

    Returns True iff:
      * header parses,
      * `|now - ts| <= max_age`,
      * `hmac.compare_digest` agrees with the recomputed digest.
    """
    try:
        ts, got = _parse_header(header)
    except (KeyError, ValueError):
        return False
    current = now if now is not None else int(time.time())
    if abs(current - ts) > max_age:
        return False
    expected = _digest(secret, ts, body)
    return hmac.compare_digest(expected, got)
```

## `dispatcher/retry.py`

```python
"""Retry policy: jittered exponential backoff + dead-letter queue push."""
from __future__ import annotations

import json
import random
from dataclasses import dataclass
from typing import Final

import redis.asyncio as aioredis

DLQ_STREAM: Final[str] = "settlements:dlq"


@dataclass(frozen=True)
class RetryPolicy:
    """Full-jitter exponential backoff.

    `backoff(n)` returns a delay uniformly drawn from
    ``[0, min(cap, base * 2**(n-1))]`` for 1-based attempt number `n`.
    Full jitter is the AWS-recommended default because it minimises
    synchronised retries across clients after a shared outage.
    """

    base: float = 2.0
    cap: float = 60.0
    max_attempts: int = 5

    def backoff(self, attempt: int) -> float:
        if attempt < 1:
            raise ValueError("attempt is 1-based")
        window = min(self.cap, self.base * (2 ** (attempt - 1)))
        return random.uniform(0, window)

    def exhausted(self, attempt: int) -> bool:
        return attempt >= self.max_attempts


async def dead_letter(
    redis: aioredis.Redis,
    event_id: str,
    merchant_id: str,
    reason: str,
    payload: dict | None = None,
) -> None:
    """Push an exhausted event onto the DLQ stream for operator replay."""
    await redis.xadd(
        DLQ_STREAM,
        {
            "event_id": event_id,
            "merchant_id": merchant_id,
            "reason": reason,
            "payload": json.dumps(payload or {"event_id": event_id}),
        },
    )
```

## `dispatcher/client.py`

```python
"""WebhookDispatcher: rate-limited, signed, retrying webhook delivery loop.

Components:
  * Redis stream consumer (`settlements:v2` / `dispatcher-g1`)
  * Per-merchant Postgres lookup with in-memory cache
  * Redis-backed token bucket rate limiter (survives restarts)
  * Per-merchant circuit breaker (open on 5 consecutive 5xx, half-open 30s)
  * Idempotency-key dedup via `SET NX EX=604800`
  * Jittered exponential backoff with DLQ after max_attempts
  * Prometheus metrics, structlog JSON logs, SIGTERM-safe shutdown
"""
from __future__ import annotations

import asyncio
import json
import os
import signal
import time
from contextlib import suppress
from dataclasses import dataclass
from datetime import datetime
from typing import Any, Dict, Final, Optional

import asyncpg
import httpx
import redis.asyncio as aioredis
import structlog
from prometheus_client import Counter, Histogram, start_http_server

from dispatcher.retry import RetryPolicy, dead_letter
from dispatcher.signing import sign
from events.models import SettlementEvent

# ---- logging config ---------------------------------------------------------

structlog.configure(
    processors=[
        structlog.processors.add_log_level,
        structlog.processors.TimeStamper(fmt="iso", utc=True),
        structlog.processors.JSONRenderer(),
    ],
)
log = structlog.get_logger("dispatcher")

# ---- constants --------------------------------------------------------------

STREAM: Final[str] = "settlements:v2"
GROUP: Final[str] = "dispatcher-g1"
DEFAULT_CONSUMER: Final[str] = os.environ.get("CONSUMER_ID", "worker-1")
DEDUP_PREFIX: Final[str] = "dispatcher:seen:"
DEDUP_TTL_SECONDS: Final[int] = 7 * 24 * 3600
RL_PREFIX: Final[str] = "dispatcher:rl:"

# ---- metrics ----------------------------------------------------------------

SENT = Counter(
    "webhooks_sent_total",
    "Terminal webhook dispatch outcomes",
    ["merchant_id", "outcome"],
)
LATENCY = Histogram(
    "webhook_latency_seconds",
    "HTTP POST latency for webhook delivery",
    ["merchant_id"],
    buckets=(0.05, 0.1, 0.25, 0.5, 1.0, 2.5, 5.0, 10.0),
)
RETRIES = Counter(
    "webhook_retries_total",
    "Retriable webhook failures",
    ["merchant_id", "outcome"],
)

# ---- circuit breaker --------------------------------------------------------


@dataclass
class CircuitBreaker:
    """Simple three-state breaker.

    Closed   -> normal traffic.
    Open     -> reject immediately until `reset_after` elapses.
    Half-open -> allow a single probe; first result decides next state.
    """

    failure_threshold: int = 5
    reset_after: float = 30.0
    consecutive_failures: int = 0
    opened_at: Optional[float] = None

    def record_success(self) -> None:
        self.consecutive_failures = 0
        self.opened_at = None

    def record_failure(self) -> None:
        self.consecutive_failures += 1
        if (
            self.consecutive_failures >= self.failure_threshold
            and self.opened_at is None
        ):
            self.opened_at = time.monotonic()
            log.warning("circuit_opened", threshold=self.failure_threshold)

    def allow(self) -> bool:
        if self.opened_at is None:
            return True
        # Half-open probe: re-allow one request after the reset window.
        return (time.monotonic() - self.opened_at) >= self.reset_after


# ---- token bucket -----------------------------------------------------------


class TokenBucket:
    """Redis-backed per-second fixed-window counter.

    For RPS limits this is sufficient and cheap: one `INCR` (+ one `EXPIRE`
    on window rollover) per acquired permit, shared across all replicas.
    Over-quota callers sleep until the next integer second.
    """

    def __init__(self, redis: aioredis.Redis) -> None:
        self._redis = redis

    async def acquire(self, merchant_id: str, max_rps: int) -> None:
        if max_rps <= 0:
            raise ValueError("max_rps must be positive")
        while True:
            now = int(time.time())
            key = f"{RL_PREFIX}{merchant_id}:{now}"
            count = await self._redis.incr(key)
            if count == 1:
                # window-bound key; 2s TTL covers clock skew safely
                await self._redis.expire(key, 2)
            if count <= max_rps:
                return
            # Over quota: wait for the next second boundary.
            await asyncio.sleep(max(0.005, 1.0 - (time.time() - now)))


# ---- dispatcher -------------------------------------------------------------


class WebhookDispatcher:
    """Async dispatcher service. Inject resources for tests, or let `start()`
    construct defaults from env-style URLs."""

    def __init__(
        self,
        redis_url: str,
        db_url: str,
        concurrency: int = 32,
        retry_policy: RetryPolicy | None = None,
        *,
        consumer_id: str = DEFAULT_CONSUMER,
        redis: aioredis.Redis | None = None,
        db: asyncpg.Pool | None = None,
        http: httpx.AsyncClient | None = None,
    ) -> None:
        self.redis_url = redis_url
        self.db_url = db_url
        self.concurrency = concurrency
        self.retry_policy = retry_policy or RetryPolicy()
        self.consumer_id = consumer_id
        self._redis = redis
        self._db = db
        self._http = http
        self._bucket: TokenBucket | None = None
        self._breakers: Dict[str, CircuitBreaker] = {}
        self._merchant_cache: Dict[str, dict] = {}
        self._stop = asyncio.Event()

    # ---- lifecycle ---------------------------------------------------------

    async def start(self) -> None:
        if self._redis is None:
            self._redis = aioredis.from_url(self.redis_url, decode_responses=True)
        if self._db is None:
            self._db = await asyncpg.create_pool(self.db_url, min_size=1, max_size=8)
        if self._http is None:
            self._http = httpx.AsyncClient(timeout=httpx.Timeout(10.0, connect=3.0))
        self._bucket = TokenBucket(self._redis)
        with suppress(Exception):
            await self._redis.xgroup_create(STREAM, GROUP, id="0", mkstream=True)
        log.info(
            "dispatcher_started",
            consumer=self.consumer_id,
            concurrency=self.concurrency,
        )

    async def stop(self) -> None:
        self._stop.set()
        if self._http is not None:
            await self._http.aclose()
        if self._db is not None:
            await self._db.close()
        if self._redis is not None:
            await self._redis.aclose()
        log.info("dispatcher_stopped")

    def request_stop(self) -> None:
        self._stop.set()

    # ---- helpers -----------------------------------------------------------

    async def _merchant(self, merchant_id: str) -> dict | None:
        cached = self._merchant_cache.get(merchant_id)
        if cached is not None:
            return cached
        assert self._db is not None
        row = await self._db.fetchrow(
            "SELECT merchant_id, url, secret, max_rps, active "
            "FROM merchant_webhooks WHERE merchant_id = $1",
            merchant_id,
        )
        if row is None:
            return None
        data = dict(row)
        self._merchant_cache[merchant_id] = data
        return data

    def _breaker(self, merchant_id: str) -> CircuitBreaker:
        cb = self._breakers.get(merchant_id)
        if cb is None:
            cb = CircuitBreaker()
            self._breakers[merchant_id] = cb
        return cb

    async def _claim_dedup(self, idem_key: str) -> bool:
        """Atomically claim an idempotency key.

        Returns True if this is the first time we've seen it (caller should
        send), False if it was already claimed (caller should drop)."""
        assert self._redis is not None
        key = DEDUP_PREFIX + idem_key
        ok = await self._redis.set(key, "1", nx=True, ex=DEDUP_TTL_SECONDS)
        return bool(ok)

    async def _release_dedup(self, idem_key: str) -> None:
        """Undo a dedup claim so the event can be retried on the next delivery.

        Called when we give up *without* acking (e.g. circuit-open path), so
        the message isn't permanently shadowed while we're unable to send."""
        assert self._redis is not None
        await self._redis.delete(DEDUP_PREFIX + idem_key)

    async def _post_once(
        self, event: SettlementEvent, merchant: dict, body: bytes
    ) -> httpx.Response:
        ts, sig = sign(merchant["secret"], body)
        headers = {
            "Content-Type": "application/json",
            "X-Plaidstone-Timestamp": ts,
            "X-Plaidstone-Signature": sig,
        }
        assert self._http is not None
        with LATENCY.labels(merchant_id=event.merchant_id).time():
            return await self._http.post(merchant["url"], content=body, headers=headers)

    @staticmethod
    def _serialize(event: SettlementEvent) -> bytes:
        return json.dumps(
            {
                "event_id": event.event_id,
                "merchant_id": event.merchant_id,
                "amount_cents": event.amount_cents,
                "currency": event.currency,
                "settled_at": event.settled_at.isoformat(),
                "idempotency_key": event.idempotency_key,
            },
            separators=(",", ":"),
        ).encode()

    # ---- core dispatch -----------------------------------------------------

    async def _dispatch(self, msg_id: str, event: SettlementEvent) -> None:
        merchant = await self._merchant(event.merchant_id)
        if merchant is None or not merchant["active"]:
            SENT.labels(merchant_id=event.merchant_id, outcome="inactive").inc()
            log.info("skip_inactive", merchant_id=event.merchant_id)
            await self._ack(msg_id)
            return

        # Dedup first: claim the idempotency key atomically.
        claimed = await self._claim_dedup(event.idempotency_key)
        if not claimed:
            SENT.labels(merchant_id=event.merchant_id, outcome="dedup").inc()
            log.info(
                "duplicate_dropped",
                event_id=event.event_id,
                merchant_id=event.merchant_id,
                idempotency_key=event.idempotency_key,
            )
            await self._ack(msg_id)
            return

        breaker = self._breaker(event.merchant_id)
        if not breaker.allow():
            SENT.labels(merchant_id=event.merchant_id, outcome="circuit_open").inc()
            log.warning(
                "circuit_open_skip",
                merchant_id=event.merchant_id,
                event_id=event.event_id,
            )
            # Release the dedup claim and do NOT ack — let the stream redeliver
            # once the breaker half-opens.
            await self._release_dedup(event.idempotency_key)
            return

        body = self._serialize(event)
        attempt = 0
        while True:
            attempt += 1
            assert self._bucket is not None
            await self._bucket.acquire(event.merchant_id, merchant["max_rps"])

            status: int | None = None
            error: str | None = None
            try:
                resp = await self._post_once(event, merchant, body)
                status = resp.status_code
            except (httpx.RequestError, asyncio.TimeoutError) as exc:
                error = repr(exc)
                log.warning(
                    "post_exception",
                    event_id=event.event_id,
                    merchant_id=event.merchant_id,
                    attempt=attempt,
                    error=error,
                )

            # Success
            if status is not None and 200 <= status < 300:
                breaker.record_success()
                SENT.labels(merchant_id=event.merchant_id, outcome="ok").inc()
                log.info(
                    "delivered",
                    event_id=event.event_id,
                    merchant_id=event.merchant_id,
                    attempt=attempt,
                    status=status,
                )
                await self._ack(msg_id)
                return

            # Classify failure
            if status is None:
                outcome = "exception"
                breaker.record_failure()
            elif 500 <= status < 600:
                outcome = "5xx"
                breaker.record_failure()
            elif 400 <= status < 500:
                outcome = "4xx"
                # 4xx is typically a bad payload or bad secret — don't trip
                # the breaker, but do not retry endlessly either.
                RETRIES.labels(merchant_id=event.merchant_id, outcome=outcome).inc()
                await self._push_dlq(event, f"client_error status={status}")
                SENT.labels(merchant_id=event.merchant_id, outcome="dlq").inc()
                await self._ack(msg_id)
                return
            else:
                outcome = str(status)
                breaker.record_failure()

            RETRIES.labels(merchant_id=event.merchant_id, outcome=outcome).inc()

            if self.retry_policy.exhausted(attempt):
                await self._push_dlq(
                    event,
                    reason=f"exhausted status={status} error={error}",
                )
                SENT.labels(merchant_id=event.merchant_id, outcome="dlq").inc()
                log.error(
                    "exhausted_to_dlq",
                    event_id=event.event_id,
                    merchant_id=event.merchant_id,
                    attempts=attempt,
                )
                await self._ack(msg_id)
                return

            delay = self.retry_policy.backoff(attempt)
            log.info(
                "retry_scheduled",
                event_id=event.event_id,
                merchant_id=event.merchant_id,
                attempt=attempt,
                delay_seconds=round(delay, 3),
                status=status,
            )
            await asyncio.sleep(delay)

    async def _push_dlq(self, event: SettlementEvent, reason: str) -> None:
        assert self._redis is not None
        await dead_letter(
            self._redis,
            event.event_id,
            event.merchant_id,
            reason,
            payload={
                "event_id": event.event_id,
                "merchant_id": event.merchant_id,
                "idempotency_key": event.idempotency_key,
            },
        )

    async def _ack(self, msg_id: str) -> None:
        assert self._redis is not None
        await self._redis.xack(STREAM, GROUP, msg_id)

    @staticmethod
    def _parse(fields: dict[str, Any]) -> SettlementEvent:
        return SettlementEvent(
            event_id=fields["event_id"],
            merchant_id=fields["merchant_id"],
            amount_cents=int(fields["amount_cents"]),
            currency=fields["currency"],
            settled_at=datetime.fromisoformat(fields["settled_at"]),
            idempotency_key=fields["idempotency_key"],
        )

    # ---- run loop ----------------------------------------------------------

    async def run(self) -> None:
        await self.start()
        sem = asyncio.Semaphore(self.concurrency)
        inflight: set[asyncio.Task[None]] = set()

        async def _guarded(msg_id: str, event: SettlementEvent) -> None:
            async with sem:
                try:
                    await self._dispatch(msg_id, event)
                except Exception as exc:  # pragma: no cover - defensive
                    log.exception(
                        "dispatch_unhandled",
                        event_id=event.event_id,
                        merchant_id=event.merchant_id,
                        error=repr(exc),
                    )

        try:
            while not self._stop.is_set():
                assert self._redis is not None
                entries = await self._redis.xreadgroup(
                    GROUP,
                    self.consumer_id,
                    {STREAM: ">"},
                    count=self.concurrency,
                    block=1000,
                )
                if not entries:
                    continue
                for _stream, msgs in entries:
                    for msg_id, fields in msgs:
                        event = self._parse(fields)
                        task = asyncio.create_task(_guarded(msg_id, event))
                        inflight.add(task)
                        task.add_done_callback(inflight.discard)
            # Graceful drain: wait for in-flight dispatches before closing pools.
            if inflight:
                log.info("draining", inflight=len(inflight))
                await asyncio.gather(*inflight, return_exceptions=True)
        finally:
            await self.stop()


# ---- entrypoint -------------------------------------------------------------


async def _amain() -> None:
    start_http_server(int(os.environ.get("METRICS_PORT", "9100")))
    d = WebhookDispatcher(
        redis_url=os.environ.get("REDIS_URL", "redis://localhost:6379/0"),
        db_url=os.environ.get("DB_URL", "postgresql://localhost/plaidstone"),
        concurrency=int(os.environ.get("CONCURRENCY", "32")),
    )
    loop = asyncio.get_running_loop()
    for sig_name in (signal.SIGTERM, signal.SIGINT):
        loop.add_signal_handler(sig_name, d.request_stop)
    await d.run()


if __name__ == "__main__":
    asyncio.run(_amain())
```

## `tests/test_retry.py`

```python
from __future__ import annotations

import pytest

from dispatcher.retry import RetryPolicy


def test_backoff_never_exceeds_cap() -> None:
    p = RetryPolicy(base=2.0, cap=10.0, max_attempts=10)
    for attempt in range(1, 11):
        for _ in range(200):
            assert 0.0 <= p.backoff(attempt) <= 10.0


def test_backoff_window_grows_exponentially() -> None:
    # Exercising the window bound: we draw many samples and assert the
    # max observed is within the theoretical window.
    p = RetryPolicy(base=1.0, cap=1000.0, max_attempts=10)
    for attempt in range(1, 6):
        window = 1.0 * (2 ** (attempt - 1))
        samples = [p.backoff(attempt) for _ in range(500)]
        assert max(samples) <= window
        # Should occasionally get close to the window ceiling.
        assert max(samples) >= window * 0.5


def test_exhausted_after_max_attempts() -> None:
    p = RetryPolicy(max_attempts=5)
    assert not p.exhausted(4)
    assert p.exhausted(5)
    assert p.exhausted(100)


def test_backoff_rejects_zero() -> None:
    with pytest.raises(ValueError):
        RetryPolicy().backoff(0)
```

```python
@pytest.mark.asyncio
async def test_token_bucket_enforces_rps_under_concurrency(fake_redis) -> None:
    """50 concurrent acquires @ 10 rps must take >= ~4s."""
    bucket = TokenBucket(fake_redis)
    start = time.monotonic()

    async def one() -> None:
        await bucket.acquire("merch_rl", max_rps=10)

    await asyncio.gather(*(one() for _ in range(50)))
    elapsed = time.monotonic() - start
    # 50 permits through a 10/sec window -> 5 full windows -> ~4s floor.
    assert elapsed >= 3.9, f"rate limit too lax: {elapsed:.2f}s"


@pytest.mark.asyncio
async def test_token_bucket_survives_restart(fake_redis) -> None:

    # per-second window in Redis and continues enforcing the limit.
    b1 = TokenBucket(fake_redis)
```
