# Plaidstone Webhook Dispatcher — v3

## `README.md`

```markdown
# Plaidstone Webhook Dispatcher

Async Python 3.11 service that consumes `SettlementEvent` messages from the
Redis stream `settlements:v2` (consumer group `dispatcher-g1`) and dispatches
signed HTTP POSTs to merchant webhook endpoints registered in the
`merchant_webhooks` Postgres table.

Filed against PLD-2147 to eliminate the duplicate-batch delivery seen after
the March 3, 2026 outage. Ship target: before April 30 quarterly close.

## Architecture

```
 ┌────────────────┐   xreadgroup     ┌────────────────────┐  POST  ┌──────────┐
 │ settlements:v2 │ ───────────────▶ │ WebhookDispatcher  │ ─────▶ │ Merchant │
 └────────────────┘                  │  rate-limit        │        └──────────┘
                                     │  HMAC sign         │
 ┌────────────────┐  xack / dedup    │  retry + breaker   │  dlq   ┌──────────┐
 │   Redis KV     │ ◀─────────────── │  idem: 7-day TTL   │ ─────▶ │ settlements:dlq │
 └────────────────┘                  └────────────────────┘        └──────────┘
```

## Guarantees

- **At-least-once delivery** with idempotency-key dedup (Redis `SET NX EX 604800`).
- **Per-merchant rate limit** enforced via Redis `INCR`/`EXPIRE` (survives
  restarts; shared across replicas).
- **Signed payloads**: `X-Plaidstone-Signature: t=<unix_ts>,v1=<hex>` over the
  raw JSON body, HMAC-SHA256. `X-Plaidstone-Timestamp` carries the same `ts`.
  Verification uses `hmac.compare_digest` and rejects events older than 5 min.
- **Exponential backoff retries** with full jitter: base 2s, cap 60s, max 5
  attempts. Exhausted retries push to `settlements:dlq` with the last status
  and failure reason.
- **Per-merchant circuit breaker**: opens after 5 consecutive 5xx responses,
  half-opens after 30s, closes on the next 2xx. Open-breaker traffic is
  DLQ'd rather than hammering the merchant.
- **Prometheus metrics** on `:9108/metrics`:
  `webhooks_sent_total{merchant_id,outcome}`,
  `webhook_latency_seconds{merchant_id}` (histogram),
  `webhook_retries_total{merchant_id,outcome}`.
- **`structlog` JSON logs** with `event_id` and `merchant_id` bound.
- **Graceful SIGTERM/SIGINT shutdown**: in-flight dispatches drain; no new
  messages are claimed from the stream.
```

## `dispatcher/signing.py`

```python
"""HMAC-SHA256 signing in Stripe-style `t=<ts>,v1=<hex>` format.

We deliberately use `hmac.compare_digest` on verify to neutralize
timing side channels, and reject any event whose timestamp is farther
than `MAX_SKEW_SECONDS` from `now` to mitigate replay attacks.
"""
from __future__ import annotations

import hmac
import time
from hashlib import sha256
from typing import Tuple

MAX_SKEW_SECONDS = 300  # 5 min — matches the iteration target


def _signed_payload(ts: int, body: bytes) -> bytes:
    return f"{ts}.".encode("ascii") + body


def sign(secret: str, body: bytes, ts: int | None = None) -> Tuple[str, int]:
    """Return `(header_value, ts)` suitable for X-Plaidstone-Signature."""
    ts = ts if ts is not None else int(time.time())
    digest = hmac.new(secret.encode(), _signed_payload(ts, body), sha256).hexdigest()
    return f"t={ts},v1={digest}", ts


def _parse_header(header: str) -> Tuple[int, str] | None:
    t: int | None = None
    v1: str | None = None
    for part in header.split(","):
        part = part.strip()
        if not part or "=" not in part:
            continue
        k, val = part.split("=", 1)
        if k == "t":
            try:
                t = int(val)
            except ValueError:
                return None
        elif k == "v1":
            v1 = val
    if t is None or v1 is None:
        return None
    return t, v1


def verify(
    secret: str,
    body: bytes,
    header: str,
    *,
    now: int | None = None,
    max_skew: int = MAX_SKEW_SECONDS,
) -> bool:
    """Constant-time verify against `header`. Returns False on any failure."""
    parsed = _parse_header(header)
    if parsed is None:
        return False
    ts, provided = parsed
    now = now if now is not None else int(time.time())
    if abs(now - ts) > max_skew:
        return False
    expected = hmac.new(secret.encode(), _signed_payload(ts, body), sha256).hexdigest()
    return hmac.compare_digest(expected, provided)
```

## `dispatcher/retry.py`

```python
"""Retry policy and dead-letter-queue publisher."""
from __future__ import annotations

import json
import random
from dataclasses import asdict, dataclass
from typing import Optional

import redis.asyncio as redis

from events.models import SettlementEvent

DLQ_STREAM = "settlements:dlq"


@dataclass(frozen=True, slots=True)
class RetryPolicy:
    """Exponential backoff with full jitter.

    Attempts are 1-indexed. `delay(1)` is in `[0, base)`, `delay(2)` is in
    `[0, min(base*2, cap))`, etc.
    """

    base_seconds: float = 2.0
    cap_seconds: float = 60.0
    max_attempts: int = 5

    def delay(self, attempt: int) -> float:
        if attempt < 1:
            return 0.0
        raw = self.base_seconds * (2 ** (attempt - 1))
        return random.uniform(0.0, min(raw, self.cap_seconds))

    def should_retry(self, attempt: int) -> bool:
        return attempt < self.max_attempts


async def push_dlq(
    client: "redis.Redis",
    event: SettlementEvent,
    reason: str,
    last_status: Optional[int] = None,
) -> None:
    """Append a failure record to `settlements:dlq` for manual review."""
    payload = asdict(event)
    payload["settled_at"] = event.settled_at.isoformat()
    await client.xadd(
        DLQ_STREAM,
        {
            "event": json.dumps(payload, separators=(",", ":")),
            "reason": reason,
            "last_status": "" if last_status is None else str(last_status),
        },
    )
```

## `dispatcher/ratelimit.py`

```python
"""Redis-backed per-merchant rate limiter.

Uses a fixed-window `INCR`+`EXPIRE` counter keyed by
`rl:<merchant>:<unix_second>`. The `EXPIRE` is set to 2 seconds so the key
naturally reclaims itself; `INCR` on the first hit in a new window creates
the key with TTL.

This approach is simple, restart-safe (state lives in Redis), and correct
for the per-second caps the spec calls for. True leaky-bucket behavior
would require Lua for atomicity, but we only need a ceiling: when the
window is full, callers await the next second. The cap is enforced
across all dispatcher replicas that share the same Redis.
"""
from __future__ import annotations

import asyncio
import time

import redis.asyncio as redis


class RedisRateLimiter:
    def __init__(self, client: "redis.Redis") -> None:
        self._client = client

    async def acquire(self, merchant_id: str, max_rps: int) -> None:
        if max_rps <= 0:
            raise ValueError("max_rps must be > 0")
        while True:
            now_f = time.time()
            now = int(now_f)
            key = f"rl:{merchant_id}:{now}"
            count = await self._client.incr(key)
            if count == 1:
                await self._client.expire(key, 2)
            if count <= max_rps:
                return
            # Wait until the next second window opens.
            sleep_for = max(0.0, (now + 1) - time.time())
            await asyncio.sleep(sleep_for)
```

## `dispatcher/breaker.py`

```python
"""Per-merchant in-process circuit breaker.

State is intentionally in-process rather than in Redis: the breaker
protects *this* dispatcher replica from pummeling a downed merchant. Each
replica independently discovers the outage and opens its own breaker.
"""
from __future__ import annotations

import time
from dataclasses import dataclass, field
from enum import Enum


class State(str, Enum):
    CLOSED = "closed"
    OPEN = "open"
    HALF_OPEN = "half_open"


@dataclass
class Breaker:
    threshold: int = 5
    cooldown_s: float = 30.0
    state: State = State.CLOSED
    consecutive_5xx: int = 0
    opened_at: float = 0.0
    # keep a monotonic source for tests:
    _now: callable = field(default=time.monotonic, repr=False, compare=False)

    def allow(self) -> bool:
        if self.state is State.OPEN:
            if self._now() - self.opened_at >= self.cooldown_s:
                self.state = State.HALF_OPEN
                return True
            return False
        return True

    def on_success(self) -> None:
        self.consecutive_5xx = 0
        self.state = State.CLOSED

    def on_5xx(self) -> None:
        self.consecutive_5xx += 1
        if self.consecutive_5xx >= self.threshold:
            self.state = State.OPEN
            self.opened_at = self._now()

    def on_4xx_or_error(self) -> None:
        # 4xx and connection errors shouldn't trip the 5xx-only counter,
        # but clients frequently want to treat errors as unhealthy. We
        # intentionally do nothing here — the retry policy handles it.
        pass
```

## `dispatcher/client.py`

```python
"""WebhookDispatcher: the main async consumer loop.

Pulls settlement events from `settlements:v2` via a consumer group,
enforces per-merchant rate limits, signs the body, POSTs with retries,
and acks the stream message only after delivery terminates (success, DLQ,
or breaker-short-circuit). No blocking I/O is performed in the hot path.
"""
from __future__ import annotations

import asyncio
import json
import os
import signal
import socket
from dataclasses import asdict
from datetime import datetime
from typing import Dict, Iterable, Mapping, Optional

import asyncpg
import httpx
import redis.asyncio as redis
import structlog

from events.models import SettlementEvent

from .breaker import Breaker
from .metrics import (
    start as start_metrics,
    webhook_latency_seconds,
    webhook_retries_total,
    webhooks_sent_total,
)
from .ratelimit import RedisRateLimiter
from .retry import RetryPolicy, push_dlq
from .signing import sign

STREAM = "settlements:v2"
GROUP = "dispatcher-g1"
DEDUP_TTL_SECONDS = 7 * 24 * 3600
BATCH_SIZE = 32
BLOCK_MS = 1000


def _configure_logging(level: str) -> None:
    structlog.configure(
        processors=[
            structlog.processors.TimeStamper(fmt="iso"),
            structlog.processors.add_log_level,
            structlog.processors.EventRenamer("msg"),
            structlog.processors.JSONRenderer(),
        ],
        wrapper_class=structlog.make_filtering_bound_logger(
            getattr(__import__("logging"), level.upper(), 20)
        ),
    )


log = structlog.get_logger("dispatcher")


class WebhookDispatcher:
    """Owns the async loop that drains the Redis stream.

    Dependencies are injected so the test suite can substitute fakeredis,
    an in-memory asyncpg mock, and respx-mounted httpx clients.
    """

    def __init__(
        self,
        redis_client: "redis.Redis",
        db_pool: "asyncpg.Pool",
        http_client: httpx.AsyncClient,
        *,
        consumer_id: str = "c1",
        concurrency: int = 32,
        retry_policy: Optional[RetryPolicy] = None,
    ) -> None:
        self._redis = redis_client
        self._db = db_pool
        self._http = http_client
        self._consumer = consumer_id
        self._sem = asyncio.Semaphore(concurrency)
        self._limiter = RedisRateLimiter(redis_client)
        self._retry = retry_policy or RetryPolicy()
        self._breakers: Dict[str, Breaker] = {}
        self._stop = asyncio.Event()

    # ------------------------------------------------------------------ lifecycle
    async def ensure_group(self) -> None:
        try:
            await self._redis.xgroup_create(STREAM, GROUP, id="0", mkstream=True)
        except redis.ResponseError as e:
            if "BUSYGROUP" not in str(e):
                raise

    def request_stop(self) -> None:
        self._stop.set()

    # ------------------------------------------------------------------ helpers
    def _breaker(self, merchant_id: str) -> Breaker:
        b = self._breakers.get(merchant_id)
        if b is None:
            b = Breaker()
            self._breakers[merchant_id] = b
        return b

    async def _lookup_merchant(self, merchant_id: str) -> Optional[Mapping]:
        row = await self._db.fetchrow(
            "SELECT merchant_id, url, secret, max_rps, active "
            "FROM merchant_webhooks WHERE merchant_id = $1",
            merchant_id,
        )
        return dict(row) if row else None

    async def _claim_idempotency(self, idem_key: str) -> bool:
        """Return True if we won the race to deliver; False if it's a duplicate."""
        won = await self._redis.set(
            f"idem:{idem_key}", "1", nx=True, ex=DEDUP_TTL_SECONDS
        )
        return bool(won)

    @staticmethod
    def _serialize(event: SettlementEvent) -> bytes:
        payload = asdict(event)
        payload["settled_at"] = event.settled_at.isoformat()
        return json.dumps(payload, separators=(",", ":"), sort_keys=True).encode()

    @staticmethod
    def _parse(fields: Mapping[str, str]) -> SettlementEvent:
        return SettlementEvent(
            event_id=fields["event_id"],
            merchant_id=fields["merchant_id"],
            amount_cents=int(fields["amount_cents"]),
            currency=fields["currency"],
            settled_at=datetime.fromisoformat(fields["settled_at"]),
            idempotency_key=fields["idempotency_key"],
        )

    # ------------------------------------------------------------------ delivery
    async def _post(
        self, url: str, body: bytes, signature: str, ts: int
    ) -> httpx.Response:
        return await self._http.post(
            url,
            content=body,
            headers={
                "Content-Type": "application/json",
                "X-Plaidstone-Signature": signature,
                "X-Plaidstone-Timestamp": str(ts),
                "User-Agent": "plaidstone-dispatcher/3",
            },
        )

    async def deliver(self, event: SettlementEvent) -> str:
        """Deliver a single event. Returns the outcome label.

        Possible outcomes: `ok`, `duplicate`, `inactive`, `breaker_open`,
        `dlq`.
        """
        logger = log.bind(event_id=event.event_id, merchant_id=event.merchant_id)

        merchant = await self._lookup_merchant(event.merchant_id)
        if not merchant or not merchant["active"]:
            logger.info("skip_inactive")
            webhooks_sent_total.labels(event.merchant_id, "inactive").inc()
            return "inactive"

        if not await self._claim_idempotency(event.idempotency_key):
            logger.info("duplicate_dropped", idem=event.idempotency_key)
            webhooks_sent_total.labels(event.merchant_id, "duplicate").inc()
            return "duplicate"

        breaker = self._breaker(event.merchant_id)
        if not breaker.allow():
            logger.warning("breaker_open")
            webhooks_sent_total.labels(event.merchant_id, "breaker_open").inc()
            await push_dlq(self._redis, event, "breaker_open")
            return "breaker_open"

        body = self._serialize(event)
        attempt = 0
        last_status: Optional[int] = None
        while True:
            attempt += 1
            await self._limiter.acquire(event.merchant_id, int(merchant["max_rps"]))
            signature, ts = sign(merchant["secret"], body)

            outcome: str
            try:
                with webhook_latency_seconds.labels(event.merchant_id).time():
                    resp = await self._post(merchant["url"], body, signature, ts)
                last_status = resp.status_code
                if 200 <= resp.status_code < 300:
                    breaker.on_success()
                    webhooks_sent_total.labels(event.merchant_id, "ok").inc()
                    logger.info("delivered", status=resp.status_code, attempt=attempt)
                    return "ok"
                if 500 <= resp.status_code < 600:
                    breaker.on_5xx()
                    outcome = "5xx"
                else:
                    outcome = "4xx"
            except httpx.HTTPError as e:
                logger.warning("http_error", error=str(e), attempt=attempt)
                outcome = "error"

            webhook_retries_total.labels(event.merchant_id, outcome).inc()

            if not self._retry.should_retry(attempt):
                await push_dlq(self._redis, event, outcome, last_status)
                webhooks_sent_total.labels(event.merchant_id, "dlq").inc()
                logger.error("dlq", reason=outcome, last_status=last_status)
                return "dlq"

            await asyncio.sleep(self._retry.delay(attempt))

    # ------------------------------------------------------------------ run loop
    async def _guarded(self, event: SettlementEvent) -> None:
        async with self._sem:
            try:
                await self.deliver(event)
            except Exception as e:  # noqa: BLE001 — keep the loop alive
                log.error(
                    "deliver_crashed",
                    event_id=event.event_id,
                    merchant_id=event.merchant_id,
                    error=str(e),
                )

    async def _ack_all(self, msg_ids: Iterable[str]) -> None:
        ids = list(msg_ids)
        if ids:
            await self._redis.xack(STREAM, GROUP, *ids)

    async def run(self) -> None:
        await self.ensure_group()
        log.info("dispatcher_started", consumer=self._consumer)
        while not self._stop.is_set():
            try:
                resp = await self._redis.xreadgroup(
                    GROUP, self._consumer, {STREAM: ">"},
                    count=BATCH_SIZE, block=BLOCK_MS,
                )
            except redis.ConnectionError as e:
                log.warning("redis_conn_error", error=str(e))
                await asyncio.sleep(1.0)
                continue
            if not resp:
                continue

            coros = []
            msg_ids: list[str] = []
            for _stream, messages in resp:
                for msg_id, fields in messages:
                    msg_ids.append(msg_id)
                    try:
                        event = self._parse(fields)
                    except (KeyError, ValueError) as e:
                        log.error("malformed_event", msg_id=msg_id, error=str(e))
                        continue
                    coros.append(self._guarded(event))

            if coros:
                await asyncio.gather(*coros, return_exceptions=True)
            await self._ack_all(msg_ids)

        log.info("dispatcher_stopped")


# -------------------------------------------------------------------- entrypoint
async def _main() -> None:
    _configure_logging(os.environ.get("LOG_LEVEL", "INFO"))
    redis_url = os.environ.get("REDIS_URL", "redis://redis:6379/0")
    db_url = os.environ.get("DB_URL", "postgresql://plaidstone@db/plaidstone")
    concurrency = int(os.environ.get("CONCURRENCY", "32"))
    metrics_port = int(os.environ.get("METRICS_PORT", "9108"))
    consumer_id = os.environ.get("CONSUMER_ID", socket.gethostname() or "c1")

    start_metrics(metrics_port)
    r = redis.from_url(redis_url, decode_responses=True)
    pool = await asyncpg.create_pool(db_url, min_size=1, max_size=8)

    async with httpx.AsyncClient(timeout=10.0) as http:
        dispatcher = WebhookDispatcher(
            r, pool, http,
            consumer_id=consumer_id,
            concurrency=concurrency,
        )

        loop = asyncio.get_running_loop()
        for s in (signal.SIGTERM, signal.SIGINT):
            loop.add_signal_handler(s, dispatcher.request_stop)

        try:
            await dispatcher.run()
        finally:
            await pool.close()
            await r.aclose()


if __name__ == "__main__":
    asyncio.run(_main())
```

```python
from __future__ import annotations

import itertools

import pytest

from dispatcher.breaker import Breaker, State
from dispatcher.retry import RetryPolicy


def test_retry_policy_delay_stays_within_cap():
    p = RetryPolicy(base_seconds=2.0, cap_seconds=10.0, max_attempts=5)
    for attempt in range(1, 6):
        d = p.delay(attempt)
        assert 0.0 <= d <= 10.0


def test_retry_policy_respects_max_attempts():
    p = RetryPolicy(max_attempts=3)
    assert p.should_retry(1)
    assert p.should_retry(2)
    assert not p.should_retry(3)


def test_breaker_opens_after_threshold():
    clock = itertools.count(start=1000.0, step=1.0)
    b = Breaker(threshold=5, cooldown_s=30.0, _now=lambda: next(clock))
    for _ in range(4):
        b.on_5xx()
        assert b.state is State.CLOSED
    b.on_5xx()
    assert b.state is State.OPEN


def test_breaker_half_opens_after_cooldown():
    t = [1000.0]
    b = Breaker(threshold=2, cooldown_s=30.0, _now=lambda: t[0])
    b.on_5xx(); b.on_5xx()
    assert b.state is State.OPEN
    assert not b.allow()
    t[0] += 31.0
    assert b.allow()
    assert b.state is State.HALF_OPEN


def test_breaker_closes_on_success_from_half_open():
    t = [1000.0]
    b = Breaker(threshold=2, cooldown_s=30.0, _now=lambda: t[0])
    b.on_5xx(); b.on_5xx()
    t[0] += 31.0
    b.allow()
    b.on_success()
    assert b.state is State.CLOSED
```
