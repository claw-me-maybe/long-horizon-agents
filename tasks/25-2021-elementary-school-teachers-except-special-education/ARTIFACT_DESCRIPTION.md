# 2nd Grade Place Value Unit: Three-Digit Numbers with Base-Ten Manipulatives (12 Days)

## Occupation
- **SOC:** 25-2021
- **Title:** Elementary School Teachers, Except Special Education

## Scenario
Mr. Chen teaches a self-contained 2nd grade class of 22 students at Maple Creek Elementary, a Title I school in a mid-sized district. His roster includes 3 students receiving Tier 2 math intervention (Bridges Intervention), 4 multilingual learners (2 at WIDA 2, 2 at WIDA 3, home language Spanish), one student on a 504 plan for ADHD, and 2 identified as advanced (accelerated in number sense). The class meets for a 75-minute math block daily; the unit precedes the winter benchmark (iReady Diagnostic).

## Inputs
- **Standards (CCSS-Math, Grade 2):**
  - 2.NBT.A.1 (understand hundreds, tens, ones; three-digit numbers composed of bundles)
  - 2.NBT.A.1.a ("100 can be thought of as a bundle of ten tens")
  - 2.NBT.A.1.b (100, 200, ..., 900 refer to hundreds and 0 tens/ones)
  - 2.NBT.A.3 (read/write numbers to 1000 using base-ten numerals, number names, expanded form)
  - 2.NBT.A.4 (compare three-digit numbers using >, =, <)
  - Standards for Mathematical Practice 2, 4, 7 (reason abstractly, model, look for structure)
- **Prior unit data:** Students finished two-digit place value in October. Cool-down data shows 18/22 fluent with tens/ones; 4 students (including 2 Tier 2 learners) still miscount bundles of ten.
- **Curricular program:** District adopted Eureka Math / EngageNY Grade 2 Module 3. Teacher may use/adapt.
- **Manipulatives available:** Base-ten blocks (ones cubes, tens rods, hundreds flats) — class set of 500/50/10 per student pair; place-value disks; hundred charts; digit cards 0-9.
- **Vocabulary targets:** hundreds, tens, ones, digit, value, expanded form, standard form, word form, compare, greater than, less than.
- **Schedule:** 75-min block = 10-min Number Talk, 50-min core lesson, 15-min small-group rotations.

## Artifact specification
12-day unit plan using **Concrete-Representational-Abstract (CRA) progression** and a **launch / explore / discuss / summarize** lesson structure. Must reflect **Universal Design for Learning (UDL)** principles.

Required sections:
1. **Unit overview**: big ideas, essential questions in kid-friendly language ("How can grouping help us count big numbers?"), end-of-unit assessment plan.
2. **Standards map**: each lesson tagged to specific CCSS sub-standard.
3. **Daily lesson sequence (12 days)** with per-day:
   - Learning target (student-facing "I can..." statement)
   - Number Talk prompt (retrieval / reasoning routine)
   - Launch (hook / connection to prior learning)
   - Explore (hands-on task with specified manipulatives)
   - Discuss (share strategies, named participation structure — turn-and-talk, number strings)
   - Summarize (anchor-chart addition, exit ticket)
   - Small-group rotation plan (Tier 2, core, extension)
4. **Center/rotation tasks** (at least 4 reusable stations): hands-on building, game, independent practice, teacher table.
5. **Family communication letter** (short) explaining place value and one home activity.
6. **Differentiation and accommodations**:
   - Tier 2 intervention plan (pre-teach vocabulary, use disks longer, daily fluency check)
   - MLL supports (sentence frames: "The number ___ has ___ hundreds, ___ tens, ___ ones."; bilingual anchor chart; partner pairings)
   - 504 ADHD supports (movement breaks, choice boards, reduced problem count)
   - Enrichment for 2 advanced students (four-digit exploration, riddles)
7. **End-of-unit assessment** (10-15 items) with scoring guide.

Pedagogical conventions: CRA progression; Number Talks; gradual release; anchor charts; turn-and-talk; exit tickets; UDL multiple means of representation/expression.

## Output format
Single Markdown file, ~2,000-2,800 words. Include at least one ASCII sketch of an anchor chart and a table for the daily sequence.

## Iteration targets
1. **Manipulative progression** — v1 may jump from concrete to abstract too quickly; explicitly plan the representational (drawings, disks) bridge and when to remove manipulatives.
2. **Number Talk variety** — ensure Number Talks aren't all counting-by-tens; include comparing, decomposing, and estimating prompts.
3. **Exit ticket -> rotation assignment loop** — each day's exit ticket should feed into tomorrow's small-group assignments with a decision rule.
4. **Vocabulary scaffolds for MLLs** — move beyond translations; add visual supports, total physical response (TPR), and structured partner talk.
5. **Common misconceptions** — name and address specific misconceptions (e.g., "305" read as "thirty-five"; confusing digit and value).
6. **Joy / engagement** — add at least 2 games or movement-based tasks; primary-grade plans lose engagement if they are only worksheets.

## Success criteria
- CRA progression visible across the 12 days (early lessons concrete-heavy; later lessons bridge to abstract).
- Each lesson includes a specific, scorable exit ticket aligned to that day's I-can statement.
- Tier 2, MLL, 504, and enrichment supports are each addressed with at least 2 distinct named strategies.
- Unit culminates in an assessment that covers all listed standards with at least one item per sub-standard.
- At least 3 Number Talk prompts of varying structure are included.
- Family letter is at a family-readable level (no jargon), gives one concrete at-home activity.

## Scope target
At 15 minutes, v1 should contain: unit overview with essential questions, a full 12-day skeleton with I-can targets and primary manipulative named per day, draft end-of-unit assessment items listed, and an initial differentiation section. Number Talks, rotation stations, and family letter are expected to be stubs or partial, to be iterated.
